# Paint App— онлайн-редактор на Canvas (React + MobX)

Небольшое приложение в стиле Paint, написанное на **React + TypeScript**, с использованием **HTML5 Canvas**, **MobX** для стейта и **SCSS-модулей** для стилизации.

В текущей версии реализовано локальное рисование (кисть, линии, фигуры, ластик) и история действий (undo/redo) и **совместное рисование по WebSocket**..

---

## Возможности

- Рисование на Canvas:
  - **Brush** — кисть
  - **Line** — прямые линии с предпросмотром
  - **Rectangle** — прямоугольник с предпросмотром
  - **Circle** — окружность из центра с предпросмотром
  - **Eraser** — ластик на `globalCompositeOperation: destination-out`
- Управление стилем:
  - выбор **цвета** (stroke + fill) через `<input type="color">`
- История:
  - **Undo / Redo** через снапшоты холста (`toDataURL`)
- Архитектура:
  - разделение UI / стейта / инструментов
  - расширяемая система инструментов (`Tool` + наследники)

---

## Технологии

- **Vite** — сборка и dev-сервер
- **React** + **TypeScript**
- **MobX** + `mobx-react-lite` — стейт менеджмент
- **HTML5 Canvas API**
- **SCSS Modules** — стилизация компонентов
- **clsx** — удобная работа с классами
- **WebSocket** — для совместного рисования

---

## Запуск проекта

```bash
# установка зависимостей
npm install

# запуск dev-сервера
npm run dev

# сборка
npm run build

# предпросмотр собранной версии
```
